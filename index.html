Workflow

Diagnostics (default)

FAQ

Common questions about how to use the GPT safely.

Can the GPT run commands on my systems?
No. It provides guidance, commands to run, and interpretation patterns. You remain in control of execution.
Why is remediation approval-gated?
To align with enterprise change control. Diagnostics is read-only; remediation requires explicit intent and approval.
Can I paste screenshots?
Yes. Paste screenshots or text. The GPT will extract the key details and proceed with structured diagnostics.
What should I save in tickets?
Save the Diagnostics summary (symptoms, scope, tests run, interpretation). Add Remediation steps (rollback + validation) only when used.
